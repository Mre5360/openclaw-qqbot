## 📦 What is openclaw-qqbot?

## 🛠️ Basic Troubleshooting

If openclaw-qqbot does not work as expected, try these fixes:

- **Check that OpenClaw is up to date:** Older versions could cause issues.
- **Restart OpenClaw:** Sometimes the plugin needs a restart to load settings.
- **Verify QQ login info:** Make sure your QQ username and password are correct.
- **Ensure internet connection:** The bot needs online access to communicate.
- **Reinstall the plugin:** Delete old files and download a fresh copy from the releases page.
